# .NET 10 - What's new demo

Samples from David's talk "What's New in .NET 10"

This codebase is used to illustrate new features in .NET 10 and C# 14.

See [What's new in .NET 10](https://learn.microsoft.com/dotnet/core/whats-new/dotnet-10/overview?WT.mc_id=DOP-MVP-5001655) and [What's new in C# 14](https://learn.microsoft.com/en-us/dotnet/csharp/whats-new/csharp-14?WT.mc_id=DOP-MVP-5001655) for more information.
